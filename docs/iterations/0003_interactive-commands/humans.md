# 0003 – Interactive Timer Commands

## Description
While a pomodoro session is running, the user can press single keys to control the timer: pause/resume, cancel without logging, stop early with logging, or extend the current phase by 5 minutes. Pauses and extensions are reflected in the journal entry.

## Rationale
Real work is interrupted. Being able to pause, extend, or gracefully stop a session — without killing the process — makes the timer fit actual workflows instead of forcing artificial time blocks.
