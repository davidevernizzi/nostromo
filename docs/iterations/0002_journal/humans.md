# 0002 – Session Journal

## Description
After each completed pomodoro session, automatically append an entry to a local journal. Entries are written in two places: a daily log and a per-project log, both using the same format.

## Rationale
Gives the user a persistent record of what was worked on, when, and for how long — without any manual effort. The flat file format keeps it inspectable with standard tools (cat, grep, tail) and easy to extend as the program gains new fields.
