# 0004 – Compact Output with Progress Bar

## Description
Redesign the timer display to fit within 40 columns and include a simple ASCII art progress bar showing how much of the current phase has elapsed.

## Rationale
The current output line is ~60+ characters and shows no visual progress. A compact, bar-based display works better in small terminals, tmux panes, and side-by-side layouts.
