# 0001 – CLI Pomodoro Timer

## Description
A single-file, portable CLI pomodoro timer written in Python. It tracks work sessions by project and task, persists the last-used project for convenience, and supports configurable work/break durations.

## Rationale
A lightweight personal productivity tool that lives in the terminal. No GUI, no external dependencies — just a script that can be dropped on any machine with Python 3 installed.
